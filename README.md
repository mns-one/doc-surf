# Search Engine to find phrases accross Multiple Text Documents

### Repo Structure: main.java.com.mns.wordfinder
- app/App.java -- Entry point
- model/Document.java -- Stores file content
- model/WordData.java -- Stores Positional Index data
- file/FileLoader.java -- Loads all files from dir
- parser/Tokenizer.java -- Tokenizes a given String
- index/InvertedIndex.java -- Builds n Store Positional Inverted Index
- search/Query.java -- Core search logic
